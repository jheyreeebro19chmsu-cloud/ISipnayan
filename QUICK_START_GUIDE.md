## 🏆 Gamification Explained

### Points System
- **Quiz Performance**: Earn points based on your score
  - 100% = 50 points
  - 90% = 45 points
  - 80% = 40 points
  - etc.

### Level System
- **Level 1**: 0-99 points
- **Level 2**: 100-199 points
- **Level 3**: 200-299 points
- Every 100 points = 1 level up!

### Badges You Can Earn
- 🌟 **First Steps**: Complete your first lesson
- 🏆 **Quiz Master**: Get 100% on any quiz
- 📚 **Dedicated Learner**: Complete 5 lessons
- ⭐ **Fraction Expert**: Complete all grade lessons
- 💎 **Point Collector**: Earn 200+ points

## 📊 Quiz System

### Question Types
1. **Multiple Choice**: Select the correct answer
2. **True/False**: Choose true or false
3. **Text Input**: Type your answer

### Quiz Flow
1. Read the question carefully
2. Select/type your answer
3. Click "Next" to proceed
4. Click "Submit Quiz" on the last question
5. See instant results!

### Passing Requirements
- **70% or higher** = Lesson completed ✅
- **Below 70%** = Try again for improvement

### Scoring Benefits
- **First Attempt 100%**: Unlock "Quiz Master" badge
- **Multiple Attempts**: Allowed to improve your score
- **Best Score Saved**: Your highest score is recorded

## 🐛 Troubleshooting

### Can't see my data?
- Data is stored in your browser's localStorage
- Don't clear browser data if you want to keep progress
- Each browser stores data separately

### Assignment not showing?
- Make sure you assigned it to the correct student
- Check that you're logged in as the assigned student
- Verify the assignment was created successfully

### Lesson is locked?
- Complete the previous lesson first
- Lessons unlock in sequential order
- Only the first lesson starts unlocked
