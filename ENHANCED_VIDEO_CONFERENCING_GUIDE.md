# Enhanced Video Conferencing System - User Guide

## Overview

The iSipnayan Kids Learning App now includes a **fully-featured video conferencing system** with professional tools for teachers and students. All styling has been updated to use **Times New Roman font** and **Prototype CSS** styling throughout the system.

---

## 📺 Video Conferencing Features

### For Teachers

#### 1. **Session Link Sharing**
- Automatically generated unique session codes (e.g., `ROOM-ABC123XYZ`)
- One-click copy button to share links with students
- Session link displayed in the control bar for easy reference
- Students can join directly via the shared link

**How to Share:**
1. Go to "Live Class" from sidebar or navbar
2. Unique session code is auto-generated
3. Click the **Copy** button next to the session code
4. Share the code with students via email, chat, or messaging app
5. Students can use the code to join the video session

#### 2. **Whiteboard Feature**
- Professional digital whiteboard for teaching
- Shared with all participants in real-time
- Drawing tools with pen/brush functionality
- Clear button to reset the whiteboard

**How to use:**
1. Click **Whiteboard** button in control bar (green when active)
2. A modal window opens with the shared drawing area
3. Click and drag to draw on the whiteboard
4. Click **Clear** button to erase all content
5. Close by clicking the **X** button

#### 3. **Screen Sharing**
- Share your entire screen with students
- Perfect for presentations and demos
- Status indicator shows when screen is being shared
- Green highlight on button indicates active sharing

**How to use:**
1. Click **Screen Share** button in control bar
2. Button turns green when active
3. Your screen is shared with all participants
4. Click again to stop sharing

#### 4. **Camera & Microphone Controls**
- Toggle camera on/off with Video button
- Toggle microphone on/off with Mic button
- Buttons turn red when disabled
- Visual indicators for mic/camera status

**How to use:**
1. Click **Mic** button to mute/unmute audio
2. Click **Video** button to turn camera on/off
3. Red buttons indicate disabled state
4. Participants can hear/see you when buttons are active

#### 5. **Teacher-Only Controls**
- Teachers have access to all advanced features
- Whiteboard exclusively for teachers
- Screen sharing exclusively for teachers
- Session link management for inviting students

### For Students

## 🚀 Getting Started

### For Teachers: Starting a Session

1. **Click "Live Class"** from sidebar, navbar, or dashboard card
2. **Share the Session Code** with students via email/message
3. **Enable Camera & Microphone** (toggle buttons to active)
4. **Open Whiteboard** if you want to teach with drawing tools
5. **Start Screen Sharing** to show presentations or demos
6. **Monitor Participants** using the participants button
7. **Click End** when session is complete

### For Students: Joining a Session

1. **Get Session Code** from your teacher
2. **Click "Live Class"** from sidebar, navbar, or dashboard
3. **Enter Session Code** provided by teacher
4. **Enable Camera & Microphone** (optional)
5. **View Whiteboard** and teacher's screen share
6. **Participate** - Ask questions via chat or unmute to speak
7. **Leave Session** when class ends

## 💡 Best Practices

### For Effective Teaching
1. Test camera and mic before starting session
2. Use whiteboard for visual explanations and diagrams
3. Share screen for presentations and demonstrations
4. Keep participant list open to monitor attendance
5. Ensure good lighting for video quality
6. Use stable internet connection
7. Remember whiteboard is shared with all students

### For Students
1. Join a few minutes early to test connection
2. Ensure mic is muted when not speaking
3. Keep camera on if possible (helps engagement)
4. Pay attention to teacher's whiteboard and screen shares
5. Ask questions - unmute or use chat
6. Avoid background distractions and clutter
7. Use headphones for better audio quality

## 🐛 Troubleshooting

### Camera/Microphone Not Working
- Check browser permissions for camera/mic access
- Verify device is not in use by another app
- Test microphone in system settings
- Try reloading the page
- Check internet connection

### Video Freezing or Lagging
- Check internet bandwidth (5+ Mbps recommended)
- Close other bandwidth-heavy applications
- Move closer to WiFi router
- Disable video to reduce bandwidth usage
- Try connecting again

### Can't Hear Audio
- Check system volume is not muted
- Verify mic button is not muted (should be gray/active)
- Test speakers/headphones in system settings
- Check browser audio permissions
- Try different audio output device

### Connection Issues
- Verify stable internet connection
- Check session code is entered correctly
- Session may be full - ask teacher to wait for seat
- Try closing and reopening the page
- Restart browser and rejoin
