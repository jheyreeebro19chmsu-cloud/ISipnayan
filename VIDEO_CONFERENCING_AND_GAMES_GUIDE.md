# Live Video Conferencing & Educational Games - User Guide

## Overview

Your iSipnayan Learning System now includes two powerful new features:
1. **Live Video Conferencing** - Real-time synchronous teaching and learning
2. **Educational Games** - Interactive learning games with points and leaderboards

---

## 1. LIVE VIDEO CONFERENCING

### For Teachers: How to Host a Live Class

#### Step 1: Start a Live Session
1. Navigate to the **Teacher Dashboard**
2. Click the **"Start Live Class"** card (blue with video icon)
3. You'll be taken to the video conference interface

#### Step 2: Configure Your Session (Optional)
- **Session Title**: Name your class (e.g., "Grade 2 Math Lesson")
- **Description**: Add details about the lesson
- **Grade Level**: Select which grade this class is for
- **Max Participants**: Set maximum number of students
- **Allow Recording**: Enable to record the session

#### Step 3: Share with Students
- A unique **Session Code** is generated automatically
- Share this code via email or display it on the dashboard
- Students can join using this code

#### Step 4: During the Class
- **Mute/Unmute**: Toggle your microphone
- **Video On/Off**: Toggle your camera
- **Screen Share**: Click to share your screen with students (teacher-only)
- **View Participants**: See list of all students in the session
- **Chat**: Text chat with students (if enabled)

#### Step 5: End the Session
- Click the **red "End Session"** button to finish
- Optionally attach a recording URL
- All session data is saved for later review

### For Students: How to Join a Live Class

#### Method 1: Join Using Session Code
1. Navigate to **Student Dashboard**
2. Click **"Live Classes"** card
3. Enter the **Session Code** provided by your teacher
4. Click **"Join"**

#### Method 2: Click Direct Link
- If your teacher shares a direct link, click it to join immediately

#### During the Class
- Your **video/audio permission** will be requested
- You can **mute/unmute** your microphone
- **Turn camera on/off** as needed
- **View teacher's screen** and materials
- Participate in classroom activities

#### Tips for Students
- Join a few minutes early
- Ensure good lighting and quiet environment
- Keep microphone muted when not speaking
- Ask questions in the chat

## 2. EDUCATIONAL GAMES

### Available Games

#### 1. **Math Speed Challenge** 🧮
- **Objective**: Solve math problems as quickly as possible
- **Difficulty Levels**: Easy, Medium, Hard
- **Time Limits**: 30 sec, 60 sec, 120 sec
- **Operations**: Addition, Subtraction, Multiplication
- **Scoring**: Points based on difficulty × correctness

**How to Play**:
1. Select difficulty level
2. Choose game duration
3. Click "Start Game"
4. Answer each question by clicking the correct option
5. Get instant feedback
6. Complete questions before time runs out

**Tips**:
- Easier levels = faster, more attempts possible
- Harder levels = more points per correct answer
- Practice regularly to improve speed and accuracy

#### 2. **Spelling Bee** 📖
- **Objective**: Spell words correctly
- **Multiple Levels**: Grade 2 and Grade 3 vocabulary
- **Feedback**: See spelling errors in real-time
- **Goal**: Master vocabulary through repetition

#### 3. **Science Quiz Masters** 🔬
- **Objective**: Answer science questions
- **Topics**: Biology, Physics, Chemistry (age-appropriate)
- **Question Types**: Multiple choice, fill-in-the-blank
- **Difficulty**: Progressive difficulty as you advance

#### 4. **Memory Match** 🧠
- **Objective**: Find matching pairs
- **Pattern Recognition**: Improve visual memory
- **Progressive Difficulty**: More pairs as levels increase
- **Time Tracking**: See how quickly you complete levels

#### 5. **Word Builder** 📝
- **Objective**: Create words from given letters
- **Vocabulary**: Build word recognition skills
- **Levels**: From 3-letter to 8-letter words
- **Hints**: Available to help you learn

#### 6. **Multiplication Master** ✖️
- **Objective**: Master multiplication tables
- **Difficulty**: From 2×2 to 12×12
- **Timed Challenges**: Race against the clock
- **Scoring**: Accuracy + speed = higher scores

### How to Access Games

1. **Navigate to Games**
   - Student Dashboard → Click "Educational Games" card
   - Or go to `/games` directly

2. **Browse Available Games**
   - See all games with descriptions and difficulty levels
   - Filter by category (Math, Language, Science, Memory)
   - View your best score for each game

3. **Start Playing**
   - Click any game card
   - Select difficulty level
   - Click "Play Now"
   - Complete the game

### Scoring & Rewards

#### How Points are Earned
- **Easy Difficulty**: 1x base score
- **Medium Difficulty**: 2x base score
- **Hard Difficulty**: 3x base score
- **Speed Bonus**: Bonus points for completing quickly

#### Badges & Achievements
- **First Step**: Play your first game
- **Rising Star**: Play 5 games
- **Game Master**: Play 10 games
- **Point Master**: Earn 1,000 points total

#### Leaderboards
- **Game Leaderboards**: Top scorers for each game
- **Overall Leaderboard**: Combined points across all games
- **Class Rankings**: Compare with classmates
- **Monthly Challenges**: Special limited-time competitions

### Tips for Maximizing Points

1. **Play Regularly**: More games = more points
2. **Increase Difficulty**: Higher difficulties = more points per answer
3. **Improve Accuracy**: Wrong answers earn fewer points
4. **Beat Your Times**: Speed increases your total score
5. **Unlock Badges**: Badges show your achievements
6. **Study Before Playing**: Learn the material first, then play to reinforce

### Viewing Your Game Statistics

1. Click on **your profile**
2. Go to **Game Stats** section
3. View:
   - Total games played
   - Total points earned
   - Average score
   - Best score
   - Games breakdown by category
   - Earned achievements/badges

## 5. TROUBLESHOOTING

### Video Conferencing Issues

**Problem: Camera/Microphone not working**
- Check browser permissions (Settings → Privacy & Security)
- Allow access to camera and microphone
- Test devices in settings before joining

**Problem: Audio is echoing**
- Use headphones instead of speakers
- Mute other participants if they're echoing

**Problem: Can't join session**
- Verify session code is correct
- Check internet connection
- Try refreshing the page
- Use latest browser version

### Game Issues

**Problem: Game won't load**
- Clear browser cache
- Disable ad blocker
- Try different browser
- Check internet speed

**Problem: Scores not saved**
- Check internet connection
- Wait for save confirmation
- Refresh page to verify
- Contact support if issue persists

## 6. BEST PRACTICES

### For Teachers

✅ **Do's:**
- Schedule classes in advance and notify students
- Test your setup before class starts
- Keep classes to a reasonable duration
- Record classes for absent students
- Use games to make learning fun
- Review game stats to identify struggling students

❌ **Don'ts:**
- Don't schedule too many classes (student burnout)
- Don't force students to use camera if uncomfortable
- Don't ignore students who are struggling in games
- Don't make games the only assessment method

### For Students

✅ **Do's:**
- Prepare before video classes
- Join testing early to check setup
- Play games after learning topics
- Try multiple difficulty levels
- Celebrate your achievements
- Practice regularly

❌ **Don'ts:**
- Don't join sessions without permission
- Don't distract others in class
- Don't cheat on games (for practice, not scores)
- Don't spend too much time on one game
- Don't neglect regular studying
